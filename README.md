# Iniciar APP
1. **Frontend**
    - En la carpeta de ClashAPP 
    `npm install`
    `ng server`
2. **Backend**
    - En la carpeta Backend
    Crear el archivo .env con API_KEY={api_key}
    `npm install`
    `npm run devStart`
